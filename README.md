## 第一周
### 知识点
数组、链表、跳表、栈、队列、优先队列、双端队列，都是一维的数据结构



```flow
st=>start: 开始
e=>end: 结束
op=>operation: 我的操作
cond=>condition: 确认？

st->op->cond
cond(yes)->e
cond(no)->op
```
